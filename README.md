# 🚀 Automated Productivity Tracker

## 📌 Description
An automated productivity tracker designed to help users manage daily tasks, build habits, and improve efficiency. The application provides a clean and interactive interface to track progress and stay consistent.

## 🛠️ Tech Stack
* HTML
* CSS
* JavaScript

## ✨ Features
* ✅ Add, update, and delete tasks
* ✅ Track daily habits and productivity
* ✅ User-friendly and responsive UI
* ✅ Organized dashboard for better tracking

## ⚙️ Installation & Setup
1. Clone the repository:
git clone https://github.com/Arti5803/Automated--Productivity--Tracker.git

2. Navigate to the project folder:
cd Automated--Productivity--Tracker

3. Open `index.html` in your browser
